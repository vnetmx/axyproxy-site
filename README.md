# AxyProxy Switcher — Site

Landing page and privacy policy for [AxyProxy Switcher](https://github.com/vnetmx/SwitchyOmega), a Chrome extension for managing proxy profiles.

## Deploy to Cloudflare Pages

1. Fork or connect this repo in the [Cloudflare Pages dashboard](https://dash.cloudflare.com/?to=/:account/pages)
2. Set **Framework preset** → None
3. Set **Build command** → *(empty)*
4. Set **Output directory** → `/` (root)
5. Deploy ✅

## Privacy policy URL

Once deployed, the privacy policy is available at:
- `https://your-project.pages.dev/#privacy`
- `https://your-project.pages.dev/privacy` (redirects to the above)

Use this URL in the Chrome Web Store developer dashboard under **Privacy practices → Privacy policy URL**.
